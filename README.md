# RevUC-2020
Objective:
Catch all the onions before shrek finds you! If you don't find all the onions by 30 seconds, Shrek will catch you!

HMD Based for:
Oculus Rift + xbox controller

Unity Version: 2019.3.0f6
Uses XR Toolkit Unity Package
